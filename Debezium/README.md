## Resumo do Debezium

O Debezium é uma plataforma de captura de alterações de código aberto que permite capturar e transmitir alterações de dados em tempo real de bancos de dados relacionais para sistemas de destino, como data warehouses, bancos de dados NoSQL e plataformas de streaming. Ele oferece:

* **Captura de alterações em tempo real:** Captura alterações em tabelas de banco de dados em tempo real, incluindo inserções, atualizações e exclusões.
* **Conectores para vários bancos de dados:** Suporta uma ampla gama de bancos de dados relacionais, incluindo Oracle, MySQL, PostgreSQL e SQL Server.
* **Transmissão para vários destinos:** Pode transmitir alterações para vários destinos, como Apache Kafka, Amazon Kinesis e Google Cloud Pub/Sub.
* **Processamento de alterações:** Fornece APIs para processar e transformar alterações de dados antes da transmissão.
* **Escalabilidade e alta disponibilidade:** Pode ser dimensionado horizontalmente e oferece alta disponibilidade por meio de replicação.
