# Configuração do Oracle Database 

## Instalando o Docker

Realizado o download do Docker para o Macbook M1. O site do [Docker](https://www.docker.com/) você poderá fazer o download e instalado no seu Macbook.

### Configurando a Imagem do Oracle Linux 8

Você pode usar o seguinte codigos:

    > docker pull oraclelinux:8.10
    > docker imagens 
    > docker run -d -it oraclelinux:8.10  bash             
    > docker ps
    > docker exec -it d6e92ad249f6 bash
